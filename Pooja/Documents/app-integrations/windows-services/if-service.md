# If Service  

## **Description**

This interface allows users to check the status of a service before performing an action.

---
![alt text](if-service-1.png)

Select the condition to check for the service.

**Options:**

- Is running
- Is stopped
- Is installed
- Isn't installed
- Is running
- Is paused
